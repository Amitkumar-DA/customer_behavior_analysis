# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to interactive dashboard creation and business insights. The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard development in Power BI, report generation, and presentation creation.

---

## Dataset

* Imported dataset using Python.
* Explored the structure and quality of the data.
* Handled missing values, duplicates, and inconsistent records.
* Prepared the dataset for analysis and visualization.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Performed initial data inspection.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions.
* Identified trends, patterns, and outliers.
* Created visualizations for better understanding.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected data types.
* Standardized and formatted data.

### 4. SQL Analysis

* Imported cleaned data into the database.
* Executed SQL queries using:

  * PostgreSQL
  * MySQL
  * SQL Server
* Generated insights using joins, aggregations, filtering, and grouping.

### 5. Power BI Dashboard

* Connected the cleaned dataset.
* Built interactive dashboards.
* Added KPIs, charts, slicers, and filters for data exploration.

### 6. Report Creation

* Summarized key findings and business insights.
* Documented the complete analysis process.

### 7. Presentation

* Created a professional presentation using Gamma.
* Highlighted objectives, methodology, dashboard, and key insights.

## How to Run

1. Clone the repository.
2. Open the Python notebooks in Jupyter Notebook or VS Code.
3. Install the required Python libraries.
4. Run the notebooks in sequence:

   * Data Loading
   * EDA
   * Data Cleaning
5. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
6. Execute the SQL scripts.
7. Open the Power BI dashboard (`.pbix`) file.
8. Review the report and presentation for project insights.
